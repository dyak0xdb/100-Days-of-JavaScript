# Challenge: Printing even numbers

Write a program that uses a for loop to print the even numbers between 1 and 20 (i.e. 2, 4, 6, ..., 20). The output should be displayed in the console.

<details>
  <summary>Click to view solution</summary>

  ```js
for(let i = 1 ; i <= 20 ; i++){
    if( i % 2 === 0){
        console.log(i);
        
    }
}


```
