# Falsy and Truthy Values in JavaScript

## Falsy Values

These are values that are considered **false** when evaluated in a boolean context:

- `false`
- `0`
- `''` or `""` (empty string)
- `null`
- `undefined`
- `NaN`

---

## Truthy Values

These are values that are considered **true** when evaluated in a boolean context:

- Everything that is **not falsy**
- `true`
- `'0'` (0 in a string)
- `' '` (a space in a string)
- `'false'` (the string `"false"`)
- `[]` (empty array)
- `{}` (empty object)
- `function () {}` (empty function)
